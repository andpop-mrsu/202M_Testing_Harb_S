Чек-лист тестирования функции для решения уравнения ax^2 + bx + c = 0

1. a=1, b=-3, c=2
   Ожидаемый результат: [1.0, 2.0]
   Описание: Два различных вещественных корня (D > 0).

2. a=1, b=-2, c=1
   Ожидаемый результат: [1.0]
   Описание: Два совпадающих корня (D = 0).

3. a=1, b=0, c=1
   Ожидаемый результат: []
   Описание: Нет вещественных корней (D < 0).

4. a=-1, b=0, c=1
   Ожидаемый результат: [-1.0, 1.0]
   Описание: Отрицательный коэффициент a.

5. a=0, b=2, c=-4
   Ожидаемый результат: [2.0]
   Описание: Граничный случай — линейное уравнение.

6. a=0, b=0, c=5
   Ожидаемый результат: []
   Описание: Некорректное уравнение (0 = 5).
